# Ecommerce Backend

Modular monolithic e-commerce backend built with Spring Boot and REST APIs.

## Architecture
- Modular Monolith
- Clear separation of business modules
- Single deployable application

## Modules
- User
- Product
- Cart
- Order
- Inventory
- Payment

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Key Characteristics
- Domain-driven modular structure
- Module-level separation (not microservices)
- Shared database
- Internal module communication via method calls
